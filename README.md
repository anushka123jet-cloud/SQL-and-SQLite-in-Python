# 🗄️ SQL and SQLite in Python

A beginner-friendly project demonstrating how to work with **SQLite databases** using Python. This notebook covers database creation, table management, CRUD operations, SQL queries, and Python-SQL integration through the built-in `sqlite3` library.

---

## 📖 Project Overview

SQLite is a lightweight, serverless relational database management system widely used in desktop, mobile, and embedded applications. Python provides built-in support for SQLite through the `sqlite3` module, making it easy to create and manage databases without additional setup.

This project demonstrates the fundamentals of SQL database management using Python and serves as an introduction to database programming.

---

## ✨ Key Features

- 📌 Connect to an SQLite database
- 📌 Create and manage database tables
- 📌 Insert records into tables
- 📌 Retrieve data using SQL queries
- 📌 Update existing records
- 📌 Delete records from tables
- 📌 Execute SQL commands through Python
- 📌 Commit and manage database transactions
